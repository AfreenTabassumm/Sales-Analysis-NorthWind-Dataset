# Sales-Analysis-NorthWind-Dataset
This project demonstrates a complete end-to-end Business Intelligence workflow using the Northwind dataset. The objective was to transform raw transactional data into structured insights that support strategic decision-making. The project integrates SQL for data exploration, Power BI for visualization and modeling.
# Northwind Traders Data Analysis Dashboard

## Project Overview
This project presents a comprehensive Power BI dashboard for analyzing Northwind Tradersâ€™ sales performance, customer behavior, and product distribution. The dashboard provides key insights and opportunities for optimization using various KPIs and visualizations.

## Key Performance Indicators (KPIs)
The dashboard focuses on the following primary KPIs:

- **Total Orders:** Total number of orders processed (830 orders)  
- **Total Revenue:** Total revenue generated from all orders ($1.2M)  
- **Average Order Value:** Average revenue per order ($141)  
- **Number of Customers:** Total distinct customers (91)  
- **Average Discount:** Average discount applied across orders (5%)  
- **Top-Selling Product:** Product generating the highest revenue (e.g., Chai)  

## Features
- **Filter Panel:** Allows users to filter data by customer segment, country, order date, and product category  
- **Sales Trend Analysis:** Visualizes monthly and yearly sales trends to identify seasonality and growth patterns  
- **Customer Segment Analysis:** Shows revenue and order distribution for High Value, Medium Value, and Low Value customers  
- **Product Category Distribution:** Displays sales contribution across categories such as Beverages, Confections, Produce, Dairy, and Meat/Poultry  
- **Geographic Analysis:** Maps order count and revenue distribution across countries and cities  
- **Supplier & Pricing Insights:** Provides insights into supplier contributions, product pricing, and category-wise pricing trends  

## Insights and Conclusions
- Strong overall sales performance with over $1M in total revenue  
- Beverages, Confections, and Produce are the top-selling categories  
- High-value customers contribute a significant portion of revenue, emphasizing the need for focused marketing strategies  
- Certain countries and cities show higher sales volumes, indicating key markets for expansion  
- Discounts increase order volumes but need careful management to protect margins  
- Supplier pricing varies across categories, highlighting opportunities for cost optimization
